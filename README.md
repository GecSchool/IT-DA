# 유기동물 임시보호 기반 입양 매칭 서비스

---

## 서비스 개요

임시보호자가 강아지 정보를 등록하면, 입양 희망자에게 맞춤 추천을 제공하는 웹 서비스. 보호소에서 바로 입양하는 기존 방식과 달리, 임보자가 직접 키워보며 파악한 정보를 바탕으로 책임감 있는 입양 연결을 목표로 함.

---

## 핵심 플로우

> 임보자 공고 등록 → DB 저장 → 입양희망자 설문 → 점수 기반 매칭 → 추천 → 입양 신청 → 임보자 승인/거절 → 입양자 연락처 공개

---

## 회원 시스템

- 단일 계정으로 임보자/입양자 역할 둘 다 가능
- 가입 후 목적 선택 (입양 / 임보 강아지 올리기 / 둘 다)
- 정보 입력 3단계로 분리, 스킵 가능

**1단계** 이메일, 비밀번호, 닉네임

**2단계** 거주형태, 가족구성, 아이 여부, 다른 반려동물 여부, 활동량 (추천 필수 정보)

**3단계** 원하는 강아지 크기/나이/성격 등 선호도 (스킵 가능)

---

## 임보자 기능

강아지 공고 등록 시 입력 항목:

기본정보 → 이름, 나이, 성별, 품종, 지역, 크기, 몸무게

성격 → 성격 태그 (활발함/매교많음/장난기 등)

생활패턴 → 산책량, 혼자있는 시간, 배변훈련 여부, 짖음 정도

환경 → 아파트 가능, 아이와 생활 가능, 다른 개/고양이와 합사 가능

건강 → 중성화 여부, 접종 완료 여부, 지병/복약 여부

기타 → 임보자가 선호하는 입양자 조건, 한마디, 사진 여러 장

---

## 입양희망자 기능

- 회원가입 시 입력한 환경정보 + 원하는 강아지 조건 기반으로 점수 계산
- 맞춤 강아지 추천 목록 제공
- 공고 상세 조회 (강아지 정보 + 임보자 한마디 + 사진)
- 입양 신청 버튼 → 임보자에게 알림

---

## 신청/매칭 프로세스

입양 신청 → 임보자 알림 → 임보자 승인/거절 → 입양자 연락처 공개
